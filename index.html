Anagram website

This anagram application can be used anywhere,even in your browser. You can find the docs for the api bellow.





GET Syntax

You can use the get method in this format at the back of the link, option=x&input=y .The options for x can be 1 to scramble or 2 to unscramble. The options for y can be any valid scrambled words or just words, like the word hello,or the scrambled version of hello.


The example of the syntax is http://localhost:5000/api/v1/option=1&input=hello
